## 安装
```shell
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```
此命令下载一个脚本（rustup-init，即`rustup`的安装程序）并执行，先完成`rustup`（管理 Rust 版本和相关工具的命令行工具）安装，再通过`rustup`安装`cargo`、`rustc`等工具，`cargo`、`rustc`等命令会安装到 `~/.cargo` 目录，rustup 本身的 metadata 和工具链会安装到 `~/.rustup`目录。

然后需要配置环境变量，rustup执行完，会提示让我们执行`~/.cargo/env`，这个脚本就是添加环境变量配置到`~/.profile`文件中。

## 简单编译执行
```rust
fn main() {
    println!("Hello, world!");
}
```

类似C语言编译：
```shell
$ rustc main.rs
$ ./main
Hello, world!
```

## 使用cargo创建项目
```shell
cargo new hello-rust
```

项目结构：
```
hello-rust
|- Cargo.toml
|- src
  |- main.rs
```
Cargo.toml 为 Rust 的清单文件。其中包含了项目的元数据和依赖库。

其他常用命令：
* `cargo build` 构建（发布版本，执行`cargo build --release`）
* `cargo run` 构建并执行
* `cargo check` 仅检查代码可正常通过编译
